# Pizza Sales Data Analysis (Power BI & SQL)

## Project Overview
This project focuses on analyzing pizza sales data using **SQL** for data analysis and **Power BI** for building interactive dashboards.  
The goal is to understand sales performance, customer ordering behavior, best & worst selling pizzas, and key business trends over time.

---

## Tools & Technologies Used
- SQL (for data cleaning, transformation, and analysis)
- Power BI (for data visualization & dashboards)
- Git & GitHub (for version control and project sharing)

---

## Dashboard Highlights

### Key Performance Indicators (KPIs)
- **Total Revenue:** 817.86K  
- **Total Orders:** 21,350  
- **Total Pizzas Sold:** 49,574  
- **Average Order Value:** 38.31  
- **Average Pizzas per Order:** 2.32  

---

### Sales Trends Analysis
- **Daily Trend:** Orders are highest on **Friday and Saturday**  
- **Monthly Trend:** Peak sales observed during **July, May, and August**  
- Clear seasonality and weekend demand patterns identified

---

### Category & Size Performance
- **Classic pizza category** contributes the highest sales and total orders
- **Large size pizzas** generate maximum revenue
- Category-wise and size-wise contribution analyzed using donut charts

---

### Best & Worst Sellers Analysis
####  Best Sellers
- **The Thai Chicken Pizza** – Highest revenue
- **The Classic Deluxe Pizza** – Highest quantity sold & total orders

####  Worst Sellers
- **The Brie Carre Pizza** – Lowest revenue, quantity, and total orders
- Identified underperforming pizzas for business improvement strategies

---

## Business Insights
- Weekend evenings are the busiest order times
- Certain pizza categories consistently outperform others
- Product-level analysis helps identify top revenue generators
- Insights can help optimize menu, pricing, and promotional strategies
